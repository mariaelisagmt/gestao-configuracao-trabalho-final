## Variáveis de Ambiente

- Nome do ambiente: `APP_ENV`
- Nome do aplicativo: `APP_NAME`
- Servidor do banco de dados: `DB_HOST`
- Usuário do banco de dados: `DB_USER`
- Senha do banco de dados: `DB_PASS`
